# Portfo

This repository contains a **practice project** created while following the **Zero To Mastery (ZTM)** course.

The purpose of this project is to learn and practice the fundamentals of building a personal portfolio website, including page structure, styling, and basic front-end development concepts.

> **Note:** This is a learning exercise and is **not** my official portfolio or a production-ready project. It is kept public to document my learning journey and progress.

## Purpose

- Practice HTML, CSS, and JavaScript fundamentals
- Learn how to structure a personal portfolio website
- Follow along with course materials and apply the concepts taught
- Track my progress as I continue learning web development

## Status

Completed as a course exercise. Future improvements may be added as part of continued learning.
